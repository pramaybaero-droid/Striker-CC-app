Striker community setup

Admin is now community-scoped through the players.role column. Use this page only if you want to generate a manual community seed SQL block without typing a plain PIN into source files.

The app hashes community PINs with striker_community_pin_v1|slug|pin.

Enter a 4-digit PIN.

Prefer running supabase-community-schema.sql, then:
select * from public.setup_default_iisc_community('1234');
Replace 1234 with the real PIN. That database function stores only the salted hash.
